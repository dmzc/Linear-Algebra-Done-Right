ℕ

自然数：皮亚诺公理

1889 年，意大利数学家朱塞佩·皮亚诺（Giuseppe Peano）提出了五条公理， 用最简洁的逻辑定义了自然数。这五条公理是整个数系大厦的基石——从它们出发，可以推导出自然数的加法、乘法乃至所有算术性质。

1

零的存在性

0 ∈ ℕ — 0 是一个自然数。 这是整个数系的起点，一个被公理直接断言的原始对象。

2

后继函数

存在函数 S: ℕ → ℕ， 若 n ∈ ℕ，则 S(n) ∈ ℕ。 S(n) 称为 n 的后继。

3

零非后继

∀n ∈ ℕ, S(n) ≠ 0 — 0 不是任何自然数的后继。这保证了自然数序列有起点。

4

后继的单射性

S(m) = S(n) ⇒ m = n — 不同自然数有不同后继。保证序列无环、无合并。

5

归纳公理

若 P(0) 成立，且 P(n) ⇒ P(S(n))， 则 ∀n ∈ ℕ, P(n)。 这是数学归纳法的根基。

🔢 自然数的构造

0 := ∅ 1 := S(0) = {0} 2 := S(1) = {0,1} 3 := S(2) = {0,1,2} ... n := {0,1,2,...,n−1}

在集合论中，每个自然数被定义为所有更小自然数的集合。这是冯·诺依曼的序数定义，与皮亚诺公理完全等价。

🏷️ 代数性质

加法封闭 乘法封闭 结合律 交换律 分配律 良序原理

自然数构成一个交换半环（无加法逆元，无乘法逆元）。良序原理保证每个非空子集有最小元，这是超限归纳的基础。

ℤ

整数：等价类构造

自然数中，方程 a + x = b 仅当 a ≤ b 时有解。 为了使得任意减法都有意义，我们通过自然数有序对的等价类来构造整数。 这一构造方法体现了现代代数的核心思想：用已知对象的关系来定义新对象。

⚙️ 构造步骤

1

在 ℕ × ℕ 上定义等价关系：
(a, b) ∼ (c, d) ⟺ a + d = b + c 直观上，(a, b) 代表"a − b"。

2

整数定义为等价类：
ℤ := (ℕ × ℕ) / ∼ 记 [(a, b)] 为 (a, b) 所在的等价类。

3

定义加法和乘法：
[(a,b)] + [(c,d)] := [(a+c, b+d)] [(a,b)] · [(c,d)] := [(ac+bd, ad+bc)]

4

嵌入 ℕ：n ↦ [(n, 0)]，负整数：−n ↦ [(0, n)]

💡 直观理解

有序对 (5, 2) 和 (7, 4) 等价， 因为它们都代表"3"。等价类的思想让我们用"差相等"这个关系来定义整数，而不需要先有"减法"的概念。

(5,2) ∼ (7,4) 因为 5+4 = 2+7 = 9 (3,0) ∼ (8,5) 因为 3+5 = 0+8 = 8 (0,4) ∼ (1,5) 因为 0+5 = 4+1 = 5 ← 都代表 −4

🏷️ 代数结构

加法群 交换环 整环 有序环 加法逆元

整数构成一个交换环（有加法逆元，无乘法逆元）。它是自然数半环的Grothendieck 群化。

ℚ

有理数：分式域构造

整数环中，方程 a · x = b（a ≠ 0）不一定有解。 有理数的构造是整环的分式域（field of fractions）的标准范例—— 在整数对 (分子, 分母) 上定义"交叉相乘相等"的等价关系。

⚙️ 构造步骤

1

在 ℤ × (ℤ \ {0}) 上定义等价关系：
(a, b) ∼ (c, d) ⟺ a · d = b · c 直观上，(a, b) 代表分数 a/b。

2

有理数定义为等价类：
ℚ := (ℤ × (ℤ \ {0})) / ∼ 记 a/b 为 [(a, b)] 的简写。

3

加法和乘法：
a/b + c/d := (ad + bc) / (bd) a/b · c/d := (ac) / (bd)

4

嵌入 ℤ：n ↦ n/1。每个非零元 a/b 有乘法逆 b/a。

🔍 约分与最简形式

等价关系自动处理了约分：2/4 ∼ 1/2 因为 2·2 = 4·1。 每个等价类中有唯一的最简分数（分子分母互质，分母为正）。

2/4 ∼ 3/6 ∼ 1/2 (都等价) −3/9 ∼ −1/3 (负分数) 0/5 ∼ 0/1 (零元)

🏷️ 代数结构

域 有序域 阿基米德 可数集 稠密性

有理数构成一个有序域——最小的特征为 0 的域。它在 ℝ 中稠密（任意两个实数之间有有理数），但不完备。

ℝ

实数：完备性公理

有理数虽然稠密，但存在"空隙"——例如 √2 不是有理数。 实数的构造填补了这些空隙，使数轴成为一条连续统。 两种经典构造方法殊途同归：戴德金分割与柯西序列完备化。

✂️ 戴德金分割

理查德·戴德金（1872）将实数定义为一对有理数子集 (A, B)，满足：

  • A ∪ B = ℚ，A ∩ B = ∅
  • A 和 B 均非空
  • ∀a∈A, ∀b∈B，有 a < b
  • A 无最大元
√2 的分割： A = {q ∈ ℚ | q² < 2 或 q ≤ 0} B = {q ∈ ℚ | q² > 2 且 q > 0}

若 B 有最小元，分割对应有理数；若 B 无最小元，分割对应无理数。

📈 柯西序列完备化

康托尔的方法：实数定义为有理柯西序列的等价类。

  • 序列 (aₙ) 是柯西列：∀ε>0, ∃N, ∀m,n>N: |aₘ−aₙ|<ε
  • 等价关系：(aₙ) ∼ (bₙ) ⟺ lim|aₙ−bₙ| = 0
  • ℝ := {有理柯西列} / ∼
√2 的柯西列： a₀=1, aₙ₊₁ = (aₙ + 2/aₙ)/2 → 1, 1.5, 1.416..., 1.4142..., ...

柯西列的极限自动"创造"了无理数。ℝ 是 ℚ 在通常绝对值下的完备化。

ℂ

复数：代数闭包

实数域中，方程 x² + 1 = 0 无解。 复数域 ℂ 是 ℝ 的代数闭包——它是包含 ℝ 的最小代数闭域。 构造方式极其简洁：在多项式环 ℝ[x] 中模掉不可约多项式 x²+1。

⚙️ 三种等价构造

方法一：商环构造

ℂ := ℝ[x] / (x² + 1) i := x + (x² + 1) → i² = −1

这是最"代数"的定义——ℂ 是 ℝ 的二次扩域。

方法二：有序对构造

ℂ := ℝ × ℝ (a,b) + (c,d) := (a+c, b+d) (a,b) · (c,d) := (ac−bd, ad+bc) i := (0, 1)

复数即平面向量，乘法由 i²=−1 自然导出。

方法三：矩阵表示

a + bi ↦ [[a, −b], [b, a]]

复数是形如 aI + bJ 的 2×2 实矩阵，其中 J²=−I。

👑 代数基本定理

高斯（1799）证明的代数基本定理：

任何非常数复系数多项式 P(z) = aₙzⁿ + ... + a₁z + a₀ 在 ℂ 中至少有一个根。

这意味着 ℂ 是代数闭域——任何代数方程在 ℂ 中都有解。 数系的扩张至此到达终点：ℂ 已经代数封闭，不再需要进一步扩张来解多项式方程。

🏷️ 代数结构

代数闭域 特征 0 不可序 代数完备 C/R 为 Galois

复数域不可排序（无法定义与域运算兼容的全序）， 但它是代数闭域——多项式方程求解的终极舞台。 ℂ/ℝ 是 2 次 Galois 扩张，Galois 群由复共轭生成。

🎯 数系包含链

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ

每一步扩张都严格保持了前一个数系的嵌入：ℕ ↪ ℤ ↪ ℚ ↪ ℝ ↪ ℂ。 每个嵌入都是单同态，前一个数系的运算和序关系在更大的数系中完全保留。 这是公理化构造的核心要求——扩张不是"替换"，而是"兼容地扩展"。
