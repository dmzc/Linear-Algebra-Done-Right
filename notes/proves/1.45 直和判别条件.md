# 证明1.45 直和等价条件

命题：设 $V_1,\dots,V_m$ 是 $V$ 的子空间，$S=V_1+\dots+V_m$。
$S$ 是直和 $\iff$ 满足：若 $v_1+\dots+v_m=\boldsymbol 0,\ v_k\in V_k$，则每个 $v_k=\boldsymbol 0$。

## 方向1：$S$ 是直和 $\implies$ 零分解唯一（必要性）

直和定义：$S$ 中任意向量的分解 $x=v_1+\dots+v_m,\ v_k\in V_k$ 唯一。
取 $x=\boldsymbol 0$，显然有一组分解：
$$\boldsymbol 0 = \boldsymbol 0+\boldsymbol 0+\dots+\boldsymbol 0,\quad \boldsymbol 0\in V_k$$
由分解唯一性，**不存在其他分解**。
也就是说：只要 $v_1+\dots+v_m=\boldsymbol 0,\ v_k\in V_k$，只能全部 $v_k=\boldsymbol 0$。必要性得证。

## 方向2：零分解唯一 $\implies S$ 是直和（充分性，就是你之前那段证明）

已知条件：
$$v_1+\dots+v_m=\boldsymbol 0,\ v_k\in V_k \implies \forall k,\ v_k=\boldsymbol 0 \tag{*}$$
要证：任意 $x\in S$ 的分解唯一。

1. 假设 $x$ 有两组分解
   
   $$
   x = v_1+v_2+\dots+v_m,\quad v_k\in V_k
   $$
   
   $$
   x = u_1+u_2+\dots+u_m,\quad u_k\in V_k
   $$

2. 两式相减
   
   $$
   (v_1+\dots+v_m)-(u_1+\dots+u_m) = x-x = \boldsymbol 0
   $$
   
   利用向量运算拆开：
   
   $$
   (v_1-u_1)+(v_2-u_2)+\dots+(v_m-u_m) = \boldsymbol 0
   $$

3. 记 $w_k = v_k - u_k$
   每个 $V_k$ 是子空间，对减法封闭：$v_k,u_k\in V_k \Rightarrow w_k\in V_k$。
   于是得到：
   
   $$
   w_1 + w_2 + \dots + w_m = \boldsymbol 0,\quad w_k\in V_k
   $$

4. 使用已知条件 (*)
   由“表示0只能全取零”，得：
   
   $$
   w_1=\boldsymbol 0,\ w_2=\boldsymbol 0,\ \dots,\ w_m=\boldsymbol 0
   $$
   
   即 $v_k - u_k = \boldsymbol 0 \implies v_k=u_k$，对所有 $k$。

5. 结论
   任意向量只有唯一分解，故 $S$ 是直和。充分性得证。
   
   ## 双向合并
   
   两边都成立，故 $V_1+\dots+V_m$ 为直和，当且仅当零向量只有全零分解。
